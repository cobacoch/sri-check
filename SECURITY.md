# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it via [GitHub Security Advisories](https://github.com/cobacoch/sri-check/security/advisories/new).

We will endeavour to respond as soon as possible and work with you to address the issue.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
